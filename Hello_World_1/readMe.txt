1. To create a java file " .java " extension is need to be attached with the file name.

2. At the very beginning, the machine has to have JVM installed.

3. Programmers write java code, it turns into Byte code using compiler (javac) to be ran on jvm.

4. Extension of the Byte code is " .class " and extension of the java code is " .java "

5. JRE = Java Runtime Environment.

6. JVM is a part of JRE.
